# Proj3

## Descrição

O Proj3 é uma aplicação desenvolvida para gerenciar e organizar projetos. Seu foco principal é proporcionar uma forma prática e eficiente de estruturar múltiplos projetos, facilitando o acompanhamento e desenvolvimento das tarefas associadas.

## Funcionalidades principais

- Organização de projetos em diretórios separados.
- Gerenciamento simplificado de múltiplos projetos.
- Estrutura modular para facilitar a manutenção e expansão futura.
